Stock - организация хранения электронных компонентов.

Альфа версия, пока просто как каркас приложения требующий рефакторинга и доработок.

#Install pkg
``cd server && npm i``
``cd client && npm i``
``npm i``

#Init db
``cd server && node init.js``

# Start application
``npm run dev``

Функциональные возможности:
- поиск компонентов по названию
- отображение списка компонентов
- открыть, изменить, добавить, удалить компонент
- загрузка файлов (пока без прикрепления к компоненту)
- навигация
- адаптивный дизайн
- своя реализация UI компонентов

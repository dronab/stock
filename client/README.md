### `npm install`

### `npm start`

### `port: 3000`

Реализованный функционал:
- Реализация своих UI компонентов
- Список компонентов
- Просмотр компонента и его свойств (9 свойств)
- Роутинг между страницами
- Изменение свойств компонента и их сохранение
- Добавление нового компонента
- Удаление компонента
- Частично реализован функционал корзины (чтобы изъять какое-то кол-во из склада сохраняя состояние)
- Всплывающие сообщения (с автозакрытием)
- Возможность загрузки файла
- Минимальная обработка ошибок
- Адаптивный дизайн
